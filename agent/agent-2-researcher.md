# Agent 2: 投研与风控研究

目标：从专业投资软件、投研流程和风险管理方法中提炼可落地需求，交给 Agent 1 编码。

## 研究方向

- 组合风险：最大回撤、VaR、CVaR、波动率、相关性、集中度、压力测试。
- 持仓动作：加仓、减仓、卖出、转仓、清仓的触发条件。
- 技术指标：趋势、动量、均线、相对强弱、波动收缩/放大。
- 基本面：收入增速、利润率、现金流、ROE/ROIC、估值分位、安全边际。
- 基金分析：净值回撤、持仓集中、风格漂移、费率、跟踪误差。
- 实时更新：行情、财报、公告、新闻舆情、汇率、基金净值的数据新鲜度。

## 输出格式

- 需求名称。
- 用户价值。
- 所需数据。
- 触发规则。
- UI 展示位置。
- 风险提示。

## 禁止项

- 不提出“保证 25% 收益”类需求。
- 不要求在数据不足时生成确定性建议。
- 不把单一技术指标作为强制交易依据。
